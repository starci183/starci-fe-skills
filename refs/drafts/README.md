# refs/drafts/ — staging for fetched UX-reference lessons

After fetching a course-website UX pattern from the web, jot the durable takeaway here as
`<topic>.md` (e.g. `gamification-streak.md`, `paywall-upgrade-nudge.md`). Keep it tight: the pattern · who
does it well · the takeaway for StarCi · what NOT to copy · the source URL · date.

Optional mid-work scratch — at the END of the task these fold into [`../CONTENT.md`](../CONTENT.md) (newest
wins) automatically and get deleted. No command.
