# journal/drafts/ — staging for new journal entries

Jot a fresh lesson here as `<kebab-name>.md` while working — what you just learned, what was chosen and
why, a gotcha to remember. Keep it short; date it so newest-wins on conflict.

Later, **`/starci-fe-update-mindset`** folds every draft here into [`../CONTENT.md`](../CONTENT.md) and
deletes it. Don't edit `CONTENT.md` directly mid-work — stage here, merge later.

> Code-rule drafts go in [`../../cannon/drafts/`](../../cannon/drafts/) instead; this folder is for the
> softer journal lessons.
